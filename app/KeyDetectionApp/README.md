# Keyword Detection App 

In the user interface, fill in:

- Device number: a unique number among 1 ~ 9.
  This number is used to choose the partition of the training dataset.
- Server IP: an IPv4 address of the computer your backend server is running on. You can probably find it in your system network settings. If you are running an app on emulator input 10.0.2.2.
- Server port: 8080.

Push the first button and load the dataset. This may take a minute.

Push the second button and start the training.
